ClickFix ~ Tame your mouse

A Problem

The average lifespan of a moderately priced computer mouse is too short. Advances in repair techniques allow computer mice to live onwards for many more years before being relegated to the parts bin. Unfortunately, these repairs require equipment, knowledge and the acceptance of some risk.

A Solution

Luckily, surgical procedures can be postponed for some mice with ClickFix. If a mouse exhibits multiple clicks when the button is only pressed once, it may be suffering from a problem that ClickFix can solve.

Get Started

To get going with ClickFix, hit "Download" below. This will take you to the releases page, where you can choose the latest executable. After all that, it's a simple matter of putting the file in a folder where you want to run it from - and then running it.
